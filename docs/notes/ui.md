# UI Development Rules

Agents must:

## Respect Focus Navigation
Every interactive element must:
- Have a defined focus target
- Support directional navigation
- Provide visible focus feedback

⸻

## Avoid Dense Layouts
Clair favors:
- Large interactive surfaces
- Clear typography hierarchy
- Visual calm

⸻

## Animation Guidelines
Animations should:
- Communicate spatial movement
- Never delay interaction
- Maintain consistent timing
