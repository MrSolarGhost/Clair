# Friends List Screen

- Shows a calm, list-based view of online friends and their current sessions.
- Controller navigation moves between friends; touch selects and brings up their status card.
- Provides quick actions for joining, messaging, or checking recent plays.

# Snapshot Cards

Clair supports a lightweight, offline social layer.

Users can:
- Generate a snapshot card representing their profile and stats
- Share it via QR code or short code
- Import snapshot cards from friends
- Browse imported cards in a gallery

Snapshot cards are static.
There are no live updates, no syncing, and no servers.

These cards are the customized profiles.
